User Guide

Learn how to capture growth ideas and turn them into copywriting or competitor analysis notes.

1
1. Save Web Clips (Selection)
Simply highlight any text on a web page, right-click, and select "Save to GrowthVault" from the context menu. This will save the text clipping to your default project (Inbox).
2
2. Save Entire Page (Popup)
Click the extension icon in your browser toolbar to open the popup interface. Here you can select a project and category (e.g. Value Proposition, Pain Point), type a quick note, and click "Save current page".
3
3. Search & Manage Clips (Library)
Open the Library to search, filter by project/type/domain, edit the text/note directly, and save changes or copy/delete them.
4
4. Structure & Export to Markdown
Click the "Export" navigation link, select a project and a markdown template (e.g., Competitor Analysis, Launch Post Draft), and copy the structured Markdown structure directly to your workspace.
