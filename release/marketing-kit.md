# SearchWise Promotion Kit

## Core Positioning

SearchWise is a local-first Chrome extension that cleans search result pages. It hides low-quality content farms, repeated mirrors, and noisy results, highlights query keywords, lets users block or allow sites with one click, and works without an account.

Main link: https://chromewebstore.google.com/detail/searchwise/mallmbpcgdlfocmigigahfibdabgokpj

Official site: https://searchwise-6na.pages.dev

Supported search engines: Google, Bing, Baidu, DuckDuckGo, Sogou, 360 Search, Yandex.

## Chinese Short Pitch

SearchWise 是一个本地优先的搜索结果净化插件。它可以隐藏内容农场、搬运站和低质量搜索结果，高亮关键词，并支持一键屏蔽或恢复站点。免费版无需账号，规则保存在浏览器本地，不插入第三方广告。

适合经常查技术资料、教程、文档、公司信息和英文资料的人。

## English Short Pitch

SearchWise is a local-first Chrome extension for cleaner search results. It hides content farms and low-signal pages, highlights keywords, and lets you block or allow sites directly from the results page. No account is required for the free version.
