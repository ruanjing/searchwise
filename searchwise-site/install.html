安装说明

SearchWise 已正式上架 Chrome 商店

点击下方按钮即可前往商店一键安装，开始享受更纯净的搜索体验。

前往 Chrome Web Store 前往 Edge 加载项

Microsoft Edge 加载项已提交审核，通过后商店链接将正式可用。

1

一键安装

在商店页面点击“添加至 Chrome”，确认权限后即可自动安装完成。

2

固定插件

点击浏览器右上角的“拼图”图标，将 SearchWise 固定在工具栏，方便随时查看拦截统计。

3

开始搜索

无需任何配置。访问 Google、Bing 或百度搜索技术关键词，SearchWise 会自动为你净化结果。
