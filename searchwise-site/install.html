安装说明

先用开发者模式试跑，发布后再换商店链接

当前阶段重点是验证功能和需求，不需要先投入服务器和复杂支付系统。

1

打开扩展管理页

在 Chrome 或 Edge 地址栏输入 chrome://extensions，打开右上角开发者模式。

2

加载本地扩展

点击“加载已解压的扩展程序”，选择项目里的 searchwise-extension 目录。

3

直接搜索测试

访问 Google、Bing、百度、DuckDuckGo、搜狗、360 搜索或 Yandex，搜索技术关键词并观察过滤效果。

4

后续发布商店

等本地版稳定后，再准备商店截图、隐私说明和压缩包，发布到 Chrome Web Store 与 Microsoft Edge Add-ons。
