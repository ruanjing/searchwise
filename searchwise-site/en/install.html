Install Guide

SearchWise is available on the Chrome Web Store

Open the store page, add the extension to Chrome, and start cleaning noisy search results.

Open Chrome Web Store
1

Add to Chrome

Click "Add to Chrome" on the store page and confirm the extension permissions.

2

Pin the extension

Use the puzzle icon in the Chrome toolbar to pin SearchWise, so you can quickly open the popup and settings.

3

Search as usual

No account is required. Visit a supported search engine such as Google, Bing, DuckDuckGo, Baidu, Sogou, 360 Search, or Yandex, and SearchWise will clean matching results locally.
