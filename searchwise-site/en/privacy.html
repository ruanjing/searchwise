Privacy Policy

SearchWise processes core cleanup locally by default

Last updated: May 8, 2026

Local Mode

When you use SearchWise without signing in, search result cleanup, custom blocked domains, allowed sites, and keyword highlighting run locally in your browser. Custom rules are stored in Chrome extension storage and are not uploaded to a server.

Optional Cloud Features

If you choose to sign in or use future Pro features, SearchWise may process account information, subscription status, synced cleanup rules, and optional AI summary requests. AI summaries are used only to generate short context for the current search results.

No Sale of Data

SearchWise does not sell user data, does not inject third-party ads into search results, and does not track users across websites.

Permission Use

The extension needs access to supported search result pages so it can read result titles, links, and snippets, then hide low-quality results or show helpful controls directly on the page.
